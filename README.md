
# DEV C PWA Workshop
--------- Description -----------

Ini adalah Project sederhana penggunaan service worker di website yang menggunakan css flexbox

--------- Cara Pemakaian --------

 1. Masuk ke Direktori yang diinginkan di command prompt atau terminal
 2. ketik `git clone https://github.com/fauzyahmad/devc-pwa.git`
 3. masuk ke direktori `web-iom/`
 4. buka file `index.html` untuk melihat hasil website
